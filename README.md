# Wireless-Comms
wireless-communication-simulations


This repository contains Python implementations of digital communication systems
developed as part of wireless communication coursework.

Projects include:

1. Digital Modulation Simulation
   - BASK
   - QPSK
   - 8-PSK
   - AWGN channel
   - BER analysis

2. OFDM and OFDMA System Simulation
   - Baseband OFDM transmitter and receiver
   - Cyclic prefix implementation
   - FFT/IFFT signal processing
   - Multi-user OFDMA system
   - BER performance evaluation

Technologies:
Python, NumPy, Matplotlib, SciPy
